# Country Road

基于 Three.js 的 3D 乡村模拟场景，包含昼夜循环、AI 驱动的角色行为、动物、建筑等元素。

## 技术栈

- Three.js + TypeScript
- Vite
- Gemini AI（角色行为决策）

## 快速开始

```bash
npm install
```

在项目根目录创建 `.env` 文件，配置所需的 API Key：

```env
VITE_GEMINI_API_KEY=your_api_key
```

启动开发服务器：

```bash
npm run dev
```

构建生产版本：

```bash
npm run build
```
